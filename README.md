# prueba-tecnica-vue

Prueba tecnica de Creación de Pantalla de Configuración de Encuesta
para esta prueba se ocupo Vue 3, vue router, Pinia, FontAwesome, node v 20.10.0

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
### Docker execution

```sh
docker compose up
```
